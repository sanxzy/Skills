# BD Skills

A collection of structured software development skills for planning, building, and reviewing features.

## Planning & Design

These skills help you think through problems before writing code.

- **bd-init-discussion** — Get relentlessly interviewed about a plan or design until every branch of the decision tree is resolved.

  ```
  npx skills@latest add sanxzy/Skills/bd-init-discussion
  ```

- **bd-generate-prd** — Create a PRD through user interview, codebase exploration, and module design, then save it locally and submit as a GitHub issue.

  ```
  npx skills@latest add sanxzy/Skills/bd-generate-prd
  ```

- **bd-generate-plan-from-prd** — Turn a PRD into a multi-phase implementation plan using tracer-bullet vertical slices.

  ```
  npx skills@latest add sanxzy/Skills/bd-generate-plan-from-prd
  ```

- **bd-request-refactor-plan** — Create a detailed refactor plan with tiny commits via user interview, then file it as a GitHub issue.

  ```
  npx skills@latest add sanxzy/Skills/bd-request-refactor-plan
  ```

## Development

These skills help you write, fix, and test code.

- **bd-implement** — Implement a single phase from a plan file. Writes production code with atomic commits per acceptance criterion.

  ```
  npx skills@latest add sanxzy/Skills/bd-implement
  ```

- **bd-tdd** — Test-driven development with a red-green-refactor loop. Builds features or fixes bugs one vertical slice at a time.

  ```
  npx skills@latest add sanxzy/Skills/bd-tdd
  ```

- **bd-fix** — Fix issues from a phase review by writing production code with atomic commits per acceptance criterion.

  ```
  npx skills@latest add sanxzy/Skills/bd-fix
  ```

## Review

- **bd-review** — Review an implementation report and verify all acceptance criteria are met based on the plan.

  ```
  npx skills@latest add sanxzy/Skills/bd-review
  ```
